计算机网络的分类

  • 根据传输技术，可分为广播式和单播式。
  • 根据规模大小，可以分为个域网，局域网，城域网，广域网和互联网。
    • 个域网用于连接距离比较小的设备，如蓝牙
    • 局域网一般用于家中或者咖啡厅
    • 城域网一般负责整个城市的网络
    • 广域网一般用于整个国家的连接

协议层次

  • 每一层的协议都虚拟的和其同等高度的协议直接交流
  • 每层仅通过下一层进行通信
  • 下一层为上一层提供服务接口
  • 信息在协议最底部以介质传输

  • 水平通信/虚拟通信：对等体之间进行的通信
  • 垂直通信/实际通信：相邻层进行的通信
  • 一个SDU可能要封装在多个PDU中传输

SDU，Service Data Unit

PDU，Protocol Data Unit

  • PDUN= PCIN + SDUN，SDUN = PDUN

参考模型

  • OSI参考模型

    • Application：应用层，提供用户需要的服务

    • Presentation：表示层从应用层接收输入或将解码的输出发送到应用层

    • Session：会话层，使我们能够建立连接并维护用户的特定会话。 它还可以验证用户的通信设备之间的安全性。

    • Transport：传输层，接受来自网络层的数据包，并将其发送到会话层。

    • Network：网络层，将数据从一个网络传输到另一网络 。 它在设备之间创建逻辑连接。

    • Data link：数据链路层，传送信息帧

    • Physical：传输信号的位表示

  • TCP/IP参考模型

    TCP (Transmission Control Protocol), 是面向有链接的协议；

    UDP (User Datagram Protocol), 是面向无连接的协议。

  • 本书采用的模型

OSI:

+ 非常有影响力的模型，概念清晰
+ 模型、协议和采用都因政治和复杂性而陷入困境

TCP/IP:

+ 非常成功的协议，运作良好并蓬勃发展
+ 弱模型
